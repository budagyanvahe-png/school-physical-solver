# school-physical-solver
It is physical calculator for school students to immediately calculate formulas for quick solving tasks in lessons.

### Problem
Student has NO TIME for calculations in school physical task, but he NEED get good mark. Also he may dont know theory of physics. 

### Solution
Create the code to make calculations immediately and accurate. Student just needs enter numbers and he will get the correct answer and if this student is initiative, he can follow the link and learn more physics.
